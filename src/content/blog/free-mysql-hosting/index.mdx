## TL;DR

무료로 16GB 메모리에 쿼리 최적화 된 MySQL을 사용할 수 있다니 이건 신이 맞습니다.  
데이터를 댓가로 파는지는 이제 조사가 필요 하겠지만..

## 우선 서비스 소개

우선 자세한건 [소개글](https://hoing.io/archives/3079)을 참고하세요.
OCI HeatWave MySQL은 기본 MySQL 스토리지에 데이터를 저장하면서, 쿼리 성능을 높이기 위해 대규모 병렬 처리 가능한 인메모리 엔진(HeatWave)을 사용합니다.
데이터가 메모리에 로딩되어 있을 경우, 해당 메모리 기반 엔진에서 쿼리를 실행해 훨씬 빠른 속도로 결과를 반환합니다.
이런 방식으로 쿼리 성능을 높이는 것은 OCI HeatWave MySQL의 가장 큰 특징입니다.

## 사용법은?

리소스 검색 -> DB 시스템 (HeatWave MySQL) -> DB 시스템 생성 -> 예산에 따라 플랜 선택 (무료 플랜 선택)

## 접속법

외부 아이피 할당이 어려워서, 서브넷에 바스티온 호스트를 만들고 접속 하셔야 합니다.
대충 프리티어 인스턴스 같은 서브넷에 두고 Tailnet으로 SNAT 설정 하는게 제일 빠를겁니다.
