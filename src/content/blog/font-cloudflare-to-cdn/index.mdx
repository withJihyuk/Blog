# TL;DR

Cloudflare Pages로 호스팅중인 블로그인데, LAX 리전에서 폰트를 받아오느라 로딩이 지연되는거 같아서 CDN 적용했더니 점수가 올라갔다.

## 그래서 어떻게 함?

```css
@import url("https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css");
```

이걸로 폰트 대체 해서 불러오고

```css
body {
  font-family: "Pretendard Variable", sans-serif;
}
```

이런식으로 폰트를 적용 해주면 된다. 이걸로 FCP 0.6초 줄였다.
내 기준 ICN 리전이 있는 어느 프로바이더에서 캐싱 되어 전달 되는걸 확인했다.
